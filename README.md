# Spring Assignment - Manas Pratap Singh

A Spring Boot project covering core Spring concepts implemented step by step.

## Tech Stack
- Java 11
- Spring Boot 2.4.x
- Maven
- IntelliJ IDEA

## Use Cases
| Branch | Description |
|--------|-------------|
| feature/UC1_HelloWorld_REST | Basic REST endpoint returning Hello message |
| feature/UC2_HelloWorld_WebPage | MVC Controller + Thymeleaf web page |
| feature/UC3_HelloWorld_REST_Full | Full REST controller with all HTTP methods |
| feature/UC4_SpringConcepts_Setup | Spring IoC container and ApplicationContext |
| feature/UC5_SpringConcepts_Logger | SLF4J logging with logback.xml |
| feature/UC6_SpringConcepts_DI | Dependency Injection with @Autowired |

## How to Run
```bash
./mvnw spring-boot:run
